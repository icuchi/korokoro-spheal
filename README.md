# korokoro-spheal 🌊

タマザラシがコロコロ画面を落ちてくるだけ、可愛いだけのChrome拡張です。

## できること

- どのWebページを開いてもタマザラシが降ってくる
- くるくる回転しながらランダムな速さで落下する
- ツールバーのアイコンからON/OFFを切り替えられる

## インストール方法

### 1. ファイルを準備する

1. このリポジトリをZIPでダウンロードして展開する
2. `setup.html` をChromeで開く
3. タマザラシの画像（`spheal.png`）を選択してボタンを押す
4. ダウンロードされた `spheal-main.js` をこのフォルダに入れる

### 2. Chromeに追加する

1. Chromeで `chrome://extensions` を開く
2. 右上の「デベロッパーモード」をONにする
3. 「パッケージ化されていない拡張機能を読み込む」でこのフォルダを選択

## 使い方

インストール後、どのページを開いてもタマザラシが降ってきます。
一時的に止めたいときはツールバーのアイコンを押してOFFにしてください。
